Goptical::Curve::Conic class reference
[Core module]

Declaration  

#include <Goptical/CurveConic>

class Conic;

Description  

This class provides a way to describe a rotationally symmetric conic curves with a given radius of curvature and deformation coefficient. The later can be provided either as Schwarzschild constant or Eccentricity value.

Fitting can be used to find best fit conic of an other rotationally symmetric curve either with fixed or free deformation parameter.

Curve::Sphere and Curve::Parabola offer most efficient implementations for special cases.
