Goptical::Design::CassegrainTelescope class reference
[Design module]

Declaration  

#include <Goptical/DesignCassegrainTelescope>

class CassegrainTelescope;

Description  

This class is a group of optical elements forming a cassegrain telescope. Optical elements are adjusted as needed from prescribed telescope parameters.

Common cassegrain types are handled.
