Goptical::Data::DiscreteSet class reference
[Core module]

Declaration  

#include <Goptical/DataDiscreteSet>

class DiscreteSet;

Description  

This class provides a one dimensional numerical data container where value pairs can be defined at arbitrary positions (x, y).

Severals interpolation algorithms are available to guess values between grid defined knots: Linear , Cubic , CubicDerivInit , CubicNormal , CubicDeriv .
