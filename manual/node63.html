Goptical::Material::Sellmeier class reference
[Core module]

Declaration  

#include <Goptical/MaterialSellmeier>

class Sellmeier;

Description  

This class models optical properties of dielectric materials with sellmeier refractive dispersion formula.

Refractive index formula is:

n = sqrt( term0 + term1 + ... + A )

with

termn = (Kn * wl^2) / (wl^2 - Ln) wl = wavelen in micrometer
