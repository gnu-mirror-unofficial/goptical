Goptical::Material::Herzberger class reference
[Core module]

Declaration  

#include <Goptical/MaterialHerzberger>

class Herzberger;

Description  

This class models optical properties of dielectric materials with herzberger refractive index dispersion formula.

Refractive index formula is:

n = A + B*wl^2 + C*wl^4 + D*wl^6 + E/(wl^2-c) + F/(wl^2-c)^2

wl = wavelen in micrometer c = 0.028 (default)
