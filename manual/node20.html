Goptical::Curve::Spline class reference
[Core module]

Declaration  

#include <Goptical/CurveSpline>

class Spline;

Description  

This class use a 1d discrete data set to define a rotationally symmetric sagitta/gradient curve. Several data interpolations algorithm are available allowing use of Smooth Cubic spline interpolation with or without prescribed derivative/gradient data. See Data::DiscreteSet .
