Goptical::Material::SellmeierMod class reference
[Core module]

Declaration  

#include <Goptical/MaterialSellmeierMod>

class SellmeierMod;

Description  

This class models optical properties of dielectric materials with modified sellmeier formulas.

Refractive index formula are:

SellMod2: n = sqrt(A + (B*wl^2)/(wl^2-C^2) + D/(wl^2-E^2))

Handbook1: n = sqrt(A + (B*wl^2) + C/(wl^2-D))

Handbook2: n = sqrt(A + (B*wl^2) + (C*wl^2)/(wl^2-D))

with wl = wavelen in micrometer
