Goptical::Curve::Grid class reference
[Core module]

Declaration  

#include <Goptical/CurveGrid>

class Grid;

Description  

This class use the Data::Grid 2d data set container to define a 3d non-symmetric curve. Several data interpolations algorithm are available allowing use of Bicubic spline interpolation with or without prescribed derivative/gradient data. .

The Curve::Spline curve model is preferred when dealing with rotationally symmetric curves.

See also Data::Grid class.
