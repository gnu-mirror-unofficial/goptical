Goptical::Data::SampleSet class reference
[Core module]

Declaration  

#include <Goptical/DataSampleSet>

class SampleSet;

Description  

This class provides a one dimensional numerical data container where sample values (y) are defined for each uniformly distributed (x) value.

Severals interpolation algorithms are available to guess values between grid defined knots: Linear , Cubic , CubicDerivInit , CubicNormal , CubicDeriv .
