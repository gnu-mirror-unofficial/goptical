Goptical::Sys::Group class reference
[Core module]

Declaration  

#include <Goptical/SysGroup>

class Group;

Description  

This class is used to form a group of optical elements. A group is a special kind of optical element which can contains other elements and sub-groups. Each group has its own local coordinate system.
