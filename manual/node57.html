Goptical::Material::DispersionTable class reference
[Core module]

Declaration  

#include <Goptical/DispersionTable>

class DispersionTable;

Description  

This class models optical properties of dispersiontable materials with table based refractive index data.

Refactive index can be defined for several wavelen, interpolation is used to compute index for any wavelen.
