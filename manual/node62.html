Goptical::Material::Schott class reference
[Core module]

Declaration  

#include <Goptical/MaterialSchott>

class Schott;

Description  

This class models optical properties of dielectric materials with schott refractive dispersion formula.

Refractive index formula is:

n = sqrt( ... + term(-2) + term(0) + term(2) + term(4) + term(6) + ... )

with

term(n) = Kn * wl^n

wl = wavelen in micrometer
