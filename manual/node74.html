Goptical::Math::Transform<3> class reference
[Core module]

Declaration  

#include <Goptical/MathTransform>

class Transform;

Description  

This class is a specialization of Transform.

This class describe linear and affine transformations in 3d space. It contains a 3x3 transformation matrix and a 3d translation vector. It can be used for translation, rotation and scaling in 3d.
