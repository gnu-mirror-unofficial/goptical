Goptical::Material::DielectricBase class reference
[Core module]

Declaration  

#include <Goptical/DielectricBase>

class DielectricBase;

Description  

This class contains pure virtuals.

This class models optical properties of dielectric materials. Transmittance values are stored as interpolated data sets. Temperature coeffiecients can also be defined.

Subclasses are availables to describe refractive index property in various ways.
